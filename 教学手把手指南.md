## 4. 四个任务到底做了什么

## Task1：画损失函数曲线
你已经在代码里实现了：
- 训练函数 `train()` 会记录每个 batch 的 loss；
- 会记录每个 epoch 的平均 loss；
- 用 `matplotlib` 画出曲线。

你需要理解：
- 曲线整体下降 = 模型在学习；
- 曲线上下波动 = 正常（每批数据不同）；
- 训练集 loss 低不代表测试准确率一定高（可能过拟合）。

## Task2：加正则化
你已经实现了两件事：
1. 在网络里加 Dropout（随机“屏蔽”部分神经元）
2. 在优化器里加 L2（`weight_decay`）

这两者目的都是：
- 别让模型只记住训练题；
- 提升在测试集的表现（泛化能力）。

## Task3：调超参数
你已经做了学习率对比实验：
- lr=0.0005
- lr=0.001
- lr=0.005

你要观察：
- 学习率太小：收敛慢；
- 学习率太大：可能不稳定；
- 合适学习率：收敛速度和准确率更平衡。

## Task4：自己设计网络
你已经实现了一个更现代的 CNN：
- 多个卷积层
- BatchNorm
- Dropout
- Adam 优化器

通常会比基础 LeNet 更准，但训练时间会更久。

## 8. 常见报错处理

### 问题1：`No module named torch`
解决：
```bash
pip install torch torchvision
```

### 问题2：下载数据集很慢
- 多等一会儿，第一次会下载
- 网络不稳定就重试运行该单元

### 问题3：`FileNotFoundError`（保存模型时）
本项目代码里已经自动 `os.makedirs(save_path, exist_ok=True)`，正常不会再报这个错。

### 问题4：跑得太慢
- 如果你电脑较弱，可先把 `num_epochs` 调小做调试（比如 1 或 2）；
- 最终提交前再按你计划的 epoch 重跑一次拿结果。
