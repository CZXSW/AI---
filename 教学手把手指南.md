# 完全新手版：一步一步运行 `project1-4.ipynb`

你现在只需要照着做，不用先懂所有原理。

## 第0步：先确认你要交什么
最后提交这3样：
1. `project1-4.ipynb`
2. `实验报告.pdf`
3. `weights` 文件夹里的 `.pth` 权重文件

---

## 第1步：安装环境（只做一次）

### 1. 打开终端（Windows 用 Anaconda Prompt 也可以）

### 2. 进入项目目录
把下面路径换成你自己电脑上的项目路径：
```bash
cd 你的项目路径/AI---
```

### 3. 安装需要的库
```bash
pip install torch torchvision matplotlib jupyter
```

---

## 第2步：启动 Notebook
在项目目录运行：
```bash
jupyter notebook
```
浏览器会自动打开页面。
然后点击 `project1-4.ipynb`。

---

## 第3步：运行整个作业（最关键）
打开 `project1-4.ipynb` 后：

### 方法A（推荐，一次跑完）
在顶部菜单点击：
**Kernel -> Restart & Run All**

### 方法B（逐格运行）
每个代码格按一次 `Shift + Enter`。

> 注意：必须从上到下运行，不能跳。

---

## 第4步：等它跑完会得到什么
1. 屏幕会打印每轮训练的 loss 和准确率。
2. 会生成模型权重文件（`.pth`），在 `weights/` 下面。
3. 你会看到损失曲线图（Task1）。

---

## 第5步：把结果填进报告
打开 `实验报告.md`，把“请替换为实际结果”的地方改成你跑出来的真实数值，然后导出 PDF（或复制到 Word 转 PDF）。

---

## 第6步：最常见问题

### 问题1：`No module named torch`
执行：
```bash
pip install torch torchvision
```

### 问题2：一直在下载数据
第一次会下载 CIFAR-10，正常，等它完成。

### 问题3：中途报错
通常是前面的格子没运行，直接用 **Kernel -> Restart & Run All** 重跑。

---

## 你现在就做这4件事
1. `cd` 到项目目录
2. 运行 `jupyter notebook`
3. 打开 `project1-4.ipynb` 后点 **Restart & Run All**
4. 跑完后更新报告并导出 PDF
