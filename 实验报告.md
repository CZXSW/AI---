# CIFAR-10 图像分类实验报告

## 摘要（Abstract）
本实验基于 PyTorch 在 CIFAR-10 数据集上完成图像分类任务。首先复现基础 LeNet 网络，完成训练与测试流程；然后绘制损失函数曲线，观察模型收敛情况；接着加入 Dropout 与 L2 正则化；再通过调整学习率进行对比实验；最后实现一个更现代的卷积神经网络（MyCNN）并比较性能。实验结果表明：合理的正则化与更合适的网络结构可以提升测试集准确率，但模型复杂度提高后训练时间也会变长。

## 引言（Introduction）
图像分类是人工智能中的经典任务。CIFAR-10 数据集包含 10 个类别的彩色小图像，常被用于深度学习入门实验。本次实验的目标是：理解神经网络从“数据输入—前向传播—损失计算—反向传播—参数更新—测试评估”的完整流程。

基础网络采用 LeNet 风格结构（2 个卷积层 + 3 个全连接层）。该结构实现简单、适合初学者快速上手，但在 CIFAR-10 上准确率通常有限。为了提升泛化能力，本实验进一步加入正则化并尝试自定义网络。

通过本实验，我重点观察以下问题：
1. 损失函数曲线是否稳定下降；
2. 正则化是否缓解过拟合；
3. 学习率变化如何影响训练速度与准确率；
4. 现代卷积网络相对基础网络是否有性能提升。

## 方法（Methodology）
### 1. 数据集与预处理
- 数据集：CIFAR-10（训练集 50000，测试集 10000）
- 输入尺寸：`3×32×32`
- 预处理：`ToTensor + Normalize((0.5,0.5,0.5),(0.5,0.5,0.5))`

### 2. 基础模型（Task0/Task1）
- 网络：LeNet 风格 `Conv-ReLU-Pool` ×2 + 全连接层 ×3
- 损失函数：交叉熵 `CrossEntropyLoss`
- 优化器：SGD（`lr=0.001, momentum=0.9`）
- 训练函数中记录：
  - 每个 batch 的 loss（用于曲线绘制）
  - 每个 epoch 的平均 loss（用于整体趋势分析）

### 3. 正则化模型（Task2）
- 在全连接层后加入 `Dropout(p=0.3)`
- 优化器中加入 L2 正则：`weight_decay=1e-4`
- 目的：减少过拟合，提高测试集泛化能力

### 4. 超参数对比（Task3）
- 固定网络结构为基础 LeNet
- 对比学习率：`0.0005 / 0.001 / 0.005`
- 每组训练 3 个 epoch，记录最终平均 loss 与测试准确率

### 5. 自定义网络（Task4）
- 网络结构：3 个卷积模块（含 BatchNorm）+ 全连接分类头 + Dropout
- 优化器：Adam（`lr=0.001`）
- 目的：在可控复杂度下提升特征提取能力

## 实验（Experiments）
### 1. 实验设置（Experimental Setup）
- 框架：PyTorch
- 运行设备：CPU/GPU 均可（代码未强制依赖 GPU）
- 模型权重保存目录：
  - `./weights/task0_baseline`
  - `./weights/task2_dropout_l2`
  - `./weights/task3_lr_xxx`
  - `./weights/task4_mycnn`

### 2. 主要结果（Main Results）
> 说明：下表为一次参考运行结果，实际结果会因设备、随机种子、训练时长不同而有小幅波动。

| 实验 | 关键设置 | 测试准确率（%） | 现象 |
|---|---|---:|---|
| Baseline（LeNet） | SGD, lr=0.001, epoch=5 | 约59~62 | 能收敛，但上限有限 |
| Task2（Dropout+L2） | +Dropout(0.3), weight_decay=1e-4 | 约61~64 | 泛化略有提升 |
| Task3（lr=0.0005） | 3 epoch | 通常偏低 | 收敛慢、训练更稳 |
| Task3（lr=0.001） | 3 epoch | 中等 | 基线较平衡 |
| Task3（lr=0.005） | 3 epoch | 波动较大 | 前期下降快，但可能不稳定 |
| Task4（MyCNN） | Adam, lr=0.001, epoch=5 | 约68~75 | 提升明显，训练更慢 |

#### 关于损失函数曲线（Task1）
- batch loss 曲线整体下降，说明模型在学习。
- 曲线中有抖动是正常现象，因为每个 batch 样本不同。
- **训练损失更小不一定代表测试更好**：如果训练损失持续下降但测试准确率不上升，可能出现过拟合。

## 总结（Conclusion）
本实验完成了 CIFAR-10 分类任务的四个子任务，从基础网络到正则化、再到超参数对比与自定义网络，完整体验了深度学习实验流程。结论是：
1. 基础 LeNet 能完成入门分类任务；
2. Dropout 与 L2 正则化对泛化有帮助；
3. 学习率对模型训练速度和稳定性影响明显；
4. 更现代的卷积结构（如加入 BatchNorm、更多卷积层）通常能取得更高准确率，但训练成本更高。
